## Aide integree ajoutee

- Ajout du composant `HelpPanel`.
- L'aide est locale, codee dans React pour cette V0.1.
- Elle affiche :
  - le titre `Iconotheque V0.1` ;
  - les actions principales ;
  - les raccourcis clavier ;
  - les garanties de securite / non-destruction ;
  - les limites connues ;
  - l'indication que les metadonnees sont stockees localement.
- Elle se ferme avec le bouton `Fermer`.
- Elle se ferme aussi avec `Echap`.
- Le panneau utilise un scroll vertical interne si necessaire.

## Menus / raccourcis ajoutes

- Ajout du menu `Aide`.
- Ajout de l'entree `Aide Iconotheque`.
- Ajout du raccourci `F1`.
- Ajout du canal IPC minimal `help:requested`.
- Ajout de `window.iconotheque.onHelpRequest(...)` via preload.

## Choix techniques

- Reutilisation du modele existant des menus Electron qui envoient un evenement IPC vers le renderer.
- API preload minimale : un seul callback d'ouverture d'aide, sans acces Node ni acces direct au disque.
- Overlay React simple et local.
- Aucun navigateur externe.
- Aucune lecture dynamique de Markdown dans cette mission.
- Aucun changement de schema SQLite.
- Aucune dependance externe ajoutee.

## Risques / limites

- Le contenu de l'aide est duplique par rapport a la documentation Markdown, car il est code dans le composant React pour rester simple.
- Les raccourcis documentes dependent du contexte de focus, notamment dans la grille, les modales et la visionneuse.
- Le lancement Electron court valide le demarrage, mais ne remplace pas une recette manuelle complete.
